# Tic-tac-toe-Game
A simple, interactive Tic-Tac-Toe Game built using HTML, CSS, and JavaScript, following the logic and approach taught by Love Babbar in the Web Dev sessions.
The game focuses on DOM manipulation, game logic building, and clean code structure.

--Features :-)

Two-player Tic-Tac-Toe (Player X vs Player O)

Detects:

Winner

Draw

Invalid moves

Real-time UI updates

Reset/Restart game button

Clean & responsive design

Easy-to-understand logic (perfect for beginners).

--Tech Stack:)

HTML5 – UI structure

CSS3 – Styling and layout

JavaScript (Vanilla JS) – Game logic + DOM updates.

---How It Works ?? :--)

Create a 3×3 grid using buttons/divs

Maintain an array of 9 positions representing the board

Alternate turns between X and O

After every move:

Check all 8 winning combinations

If matched → declare winner

If no empty cells left → declare draw

Disable board when the game ends

On Reset, clear array + reset UI

Core Concepts You Practice:

DOM manipulation

Condition checking

Array handling

Clean logic structuring

Event listeners

Updating UI using JS.
